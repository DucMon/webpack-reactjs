### run docker
```
step 1: build images
    docker build . -t react-webpack/dong-nguyen
step 2: start and run container
    docker run 8080:80 react-webpack/dong-nguyen
step 3: open browser. Enter the address "localhost:8080" to browser
FINISH
```
